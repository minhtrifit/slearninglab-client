# I. Installing tutorial

## Config .env file in root dir

```bash
VITE_API_URL=yourserverurl
VITE_CLOUDINARY_PRESETNAME=yourcdrpresetname
VITE_CLOUDINARY_APIKEY=yourcdrapikey
VITE_CLOUDINARY_CLOUDNAME=yourcdrcloudname
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run dev
```
